# 🌱 Grădina Mea | Ghid de Îngrijire a Plantelor

[![Status Proiect](https://img.shields.io/badge/Status-In%20Dezvoltare-green)](https://github.com/Lixandru-Marius)
[![Limbaj principal](https://img.shields.io/badge/Limbaj-HTML5-orange)](https://developer.mozilla.org/en-US/docs/Web/HTML)

Acesta este un proiect web personal creat pentru a împărtăși pasiunea pentru grădinărit. Site-ul oferă informații structurate despre îngrijirea plantelor, sfaturi sezoniere și tehnici de întreținere pentru o grădină sănătoasă.
